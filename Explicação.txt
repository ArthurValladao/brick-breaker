Como fiz o pause

Para fazer o Pause eu criei uma varialvel boleana chamada pausado e adicionei o valor falso a ela, após isso adicionei o keycode do botão que pausaria o jogo, adicionei uma function para mostrar a mensagem de jogo pausado e na function desenhar adicionei um if para avisar que se o jogo estiver pausado nada se move.

Fase 1: O canvas tem 350x280, A largura da barra é 75, a quantidade de blocos no total é 15

Fase 2: O canvas cresce para 450x400, A largura da barra é 60, a quantidade de blocos no total é 28 


Fase 3: O canvas fica 650x500, A largura da barra é 45, a quantidade de blocos no total é 45
